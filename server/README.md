# server

Placeholder for the Node.js + Express backend.

Proposed directories:
- `src/api/` controllers/handlers
- `src/routes/` HTTP routes
- `src/services/` business logic
- `src/models/` ORM/DB models
- `src/middleware/` Express middleware
- `src/utils/` helpers
- `src/config/` config and env loading
- `src/jobs/` background tasks/queues
- `src/events/` pub/sub or domain events
- `src/db/migrations/` schema migrations
- `src/db/seeders/` seed data
- `tests/` backend tests
- `scripts/` ops/dev scripts
